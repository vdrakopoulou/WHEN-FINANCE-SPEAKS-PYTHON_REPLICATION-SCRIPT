# WHEN-FINANCE-SPEAKS-PYTHON_REPLICATION-SCRIPT





This folder contains the replication materials for:

> Drakopoulou, V. *Algorithmic Stratification in Finance Degrees: Global Patterns of Python Adoption in University Curricula*.

All replication instructions and run commands are provided in the Jupyter notebook:

**`README.ipynb`**

Please open `README.ipynb` in Jupyter (or VS Code / Colab) and follow the steps there to:

- check the required folder structure,
- install dependencies,
- and run `replication_analysis.py` to regenerate all tables and figures from  
  `Data/Python_in_Finance_Curriculum_scored.csv`.
